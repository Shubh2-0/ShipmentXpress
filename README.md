## 🎯 About

**ShipmentXpress** is a comprehensive shipment management system built in Java. Manage packages from creation to delivery with ease!

## ✨ Features

- 📦 **Create Shipments** - Add new packages to system
- 🔍 **Track Shipments** - Real-time status updates
- ✏️ **Update Details** - Modify shipment information
- 🗑️ **Delete Shipments** - Remove completed/cancelled orders
- 📊 **Reports** - Generate shipping reports

## 🏗️ System Flow

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ Created  │───►│ Shipped  │───►│In Transit│───►│ Delivered│
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```
